# Day 57 — Markdown to PDF Converter

**120 Days of Code** | NIT Rourkela | Mechanical Engineering

## What this is
A CLI tool that converts Markdown files into styled
PDF documents — useful for notes, resumes, and reports.

## How to run
```bash
pip3 install markdown2 weasyprint
python3 converter.py
```

## Features
- Converts any .md file to a styled PDF
- Custom CSS themes (light/dark/academic)
- Syntax highlighting for code blocks
- Auto table of contents
- Batch conversion mode
